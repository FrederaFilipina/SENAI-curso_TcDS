# Criando e manuseando um projeto:

## Criar, nomear e configurar o projeto:
1. Crie uma pasta onde o seu projeto será salvo;
2. Abra o terminal dentro desta pasta:
    - Digite o comando: *`npm create vite@latest`*;
    - Escolhe o nome do projeto, vai ser criado uma pasta com esse nome;
    - Selecione o framework: *`React`* (é a biblioteca em que o projeto vai ser criado);
    - Selecione a linguagem: *`JavaScript`*;
    - Selecione: *`No`*, para não usar a versão Vite 8 beta;
    - Selecione: *`Yes`*, para ele já instalar e executar o comando: *`npm run dev`* (comando para executar o projeto em modo de desenvolvimento), nesse ponto ele já vai passar automaticamente para o terminal da pasta do projeto;
    - Clique no link que vai estar em:  ➜  Local, ele vai abrir o projeto no navegador;

## Fechando e reabrindo o projeto em modo desenvolvimento:
1.  Fechando o projeto:
    - Abra o terminal dentro da pasta do projeto;
    - No teclado de o comando: *`ctrl + c`*, isso faz com que o projeto seja parado de rodar;
2. Reabrindo o projeto:
    - Abra o terminal dentro da pasta do projeto;
    - Execute o comando: *`npm run dev`*;
    - Clique no link que vai estar em:  ➜  Local, ele vai abrir o projeto no navegador;

## Reabrindo o projeto em modo desenvolvimento após baixar o mesmo do *`Github`*;
1.  Reabrindo o projeto:
    - Abra o terminal dentro da pasta do projeto;
    - No teclado de o comando: *`npm i`*, isso faz com que o projeto seja inicializado com todas as dependências instaladas;
    - Execute o comando: *`npm run dev`*;
    - Clique no link que vai estar em:  ➜  Local, ele vai abrir o projeto no navegador;
